# Projetos em Python

Neste repositório, você encontrará uma coleção de 15 programas em Python que foram desenvolvidos como parte do meu aprendizado inicial com a linguagem. Cada programa possui uma funcionalidade específica e pode ser útil para diferentes propósitos.

## Lista de Programas

Aqui está uma lista dos programas disponíveis neste repositório:

1. **Calcula Média:** Um programa que calcula a média de um conjunto de números.

2. **Calcula Divisão:** Um programa que realiza a divisão entre dois números.

3. **Verifica CEP:** Um programa que verifica se um CEP informado possui a quantidade correta de dígitos.

4. **Verifica CPF:** Um programa que verifica se um CPF informado possui a quantidade correta de dígitos.

5. **Classifica Média:** Um programa que classifica um valor de média em conceitos (A, B, C, D, F).

6. **Calcula Desconto INSS:** Um programa que calcula o desconto do INSS com base no salário informado.

7. **Estrutura de Repetição For:** Um programa que utiliza a estrutura de repetição `for` para realizar uma tarefa específica.

8. **Estrutura de Repetição While:** Um programa que utiliza a estrutura de repetição `while` para realizar uma tarefa específica.

9. **Hello World:** O tradicional programa "Hello World" para imprimir uma mensagem na tela.

10. **Linguagem Visual com Turtle:** Um programa que utiliza a biblioteca Turtle para criar desenhos em diferentes direções.

11. **Leitura de Lista:** Um programa que lê uma lista de nomes e realiza uma determinada operação com eles.

12. **Verifica Par ou Ímpar:** Um programa que verifica se um número informado pelo usuário é par ou ímpar.

13. **Servidor e Cliente:** Um programa que consiste em um servidor (servidor.py) e um cliente (cliente.py) para se comunicarem entre si.

14. **Sniffer:** Um programa que atua como um "sniffer" de rede, capturando pacotes e extraindo informações relevantes.

15. **Minigame de Sorteio:** Um minigame que sorteia números aleatórios e permite ao usuário adivinhar o número sorteado.

## Como utilizar cada programa

Apenas execute o arquivo tendo baixado o Python.
