# Projetos em Python

Neste repositório, você encontrará uma coleção de 15 programas em Python que foram desenvolvidos como parte do meu aprendizado inicial com a linguagem. Cada programa possui uma funcionalidade específica e pode ser útil para diferentes propósitos.

## Lista de Programas

Aqui está uma lista dos programas disponíveis neste repositório:

1.  **Hello World:** O tradicional programa "Hello World" para imprimir uma mensagem na tela.

2.  **Leitura de Lista:** Um programa que lê uma lista de nomes e realiza uma determinada operação com eles.

3.  **Calcula Divisão:** Um programa que realiza a divisão entre dois números.

4.  **Estrutura de Repetição For:** Um programa que utiliza a estrutura de repetição `for` para realizar uma tarefa específica.

5.  **Estrutura de Repetição While:** Um programa que utiliza a estrutura de repetição `while` para realizar uma tarefa específica.

6.  **Calcula Média:** Um programa que calcula a média a partir de 3 valores informados pelo usuario retornando o resultado.

7.  **Classifica Média:** Um programa que classifica a média a partir de 3 valores informados pelo usuario e retorna se é bom ou não.

8.  **Verifica Par ou Ímpar:** Um programa que verifica se um número informado pelo usuário é par ou ímpar.

9.  **Verifica CEP:** Um programa que verifica se um CEP informado possui a quantidade correta de dígitos.

10. **Verifica CPF:** Um programa que verifica se um CPF informado possui a quantidade correta de dígitos.

11. **Linguagem Visual com Turtle:** Um programa que utiliza a biblioteca Turtle para criar desenhos em diferentes direções.

12. **Minigame de Sorteio:** Um minigame que sorteia números aleatórios e permite ao usuário adivinhar o número sorteado.

13. **Servidor e Cliente:** Um programa que consiste em um servidor (servidor.py) e um cliente (cliente.py) para se comunicarem entre si.

14. **Sniffer:** Um programa que atua como um "sniffer" de rede, capturando pacotes e extraindo informações relevantes.
